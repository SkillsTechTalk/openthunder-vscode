# Security Policy

## Reporting a vulnerability

If you find a security or privacy issue in the OpenThunder VS Code extension, please report
it privately. **Do not open a public issue** for security-sensitive reports.

- Email: **support@skillstechtalk.com** with subject `SECURITY: openthunder-vscode`
- Include: a description, steps to reproduce, affected version, and impact.

We will acknowledge your report and work with you on a fix and disclosure timeline.

## Scope

This repository is the OpenThunder VS Code extension, a thin client. It talks to the
OpenThunder engine over local HTTP and the OpenThunder CLI; it does not contain the analysis
engine, cloud services, or credentials. Issues in those components should still be reported to
the address above.

## Supported versions

The latest published version on the Visual Studio Marketplace is supported. Fixes are shipped
as new versions.
